# Craps

A simple console-based Craps game written in Java.

## How to run

### Method 1

1. Download `Craps.jar` from the [v1.0.0](https://github.com/EgnaBrandonAlan/Craps/releases/tag/v1.0.0) release page
2. Run:
   ```bash
   java -jar Craps.jar
   ```

### Method 2

1. Compile the source files in `src/`:
   ```bash
   javac -d out src/*.java
   ```
2. Run the compiled classes in `out/`:
   ```bash
   java -cp out Craps
   ```
